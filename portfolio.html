My work, all in one place!


From HTML & CSS to UI/UX design — here’s a peek into my creative process.

Things left Behind

A responsive web layout developed with HTML and CSS, utilizing Flexbox for structured, adaptable design. Created in Figma, the visual style features a balanced contrast of dark and white tones, emphasizing clarity, mood, and clean composition.

Aurelia Studios

A responsive portfolio website designed to reflect my style and skills. It features a fictional agency, clean layout, and works well on both web and mobile. This project highlights my focus on structure, storytelling, and thoughtful design.

Zoom | Re-design

A refreshed content card layout for Zoom’s homepage, redesigned to reduce visual clutter and enhance usability. I adjusted the border radius for a softer look, aligned buttons to the bottom for consistency, and centered key elements like titles and icons for better flow. Before and after available on Figma.

ZTC | Cañada College

As a product designer and student at Cañada College, I redesigned the Zero Textbook Cost (ZTC) page with a strong focus on user empathy. My dual perspective allowed me to identify real student pain points and create a clearer, more accessible experience.

Velvet Flame

Velvet Flame is an e-commerce homepage concept for a candle brand. Designed early in my Figma journey, this project reflects my exploration of layout, branding, and visual storytelling while learning the fundamentals of UI design.

Bloomify

Bloomify is a 50-page e-commerce app designed for selling flowers and gifts. As one of my first full-scale design projects, it includes key screens like splash, sign-in, home, favorites, and profile. Looking back, I see areas for improvement, which highlights just how much my skills have grown since then making this an important milestone in my design journey.
